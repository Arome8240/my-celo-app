# Hospital Interface Tasks & Features

Comprehensive guide for hospital partner operations in the Hospital Network Token (HNT) system.

## Overview

The hospital interface enables healthcare facilities to manage their operations within the multi-hospital network, including patient management, appointments, token rewards, and inter-facility coordination.
